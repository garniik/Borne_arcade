# Simple Pacman

Un jeu simple inspiré de Pacman créé avec pygame.

## Contrôles
- Flèches directionnelles : déplacer le joueur
- R : recommencer après game over

## Objectif
Mangez tous les points blancs tout en évitant les fantômes.

## Installation
```bash
pip install -r requirements.txt
```

## Lancement
```bash
python main.py
```
